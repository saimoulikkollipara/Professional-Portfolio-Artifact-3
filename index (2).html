Artifact-1: TimeSync Assistant: A Trust-Centered Scheduling Chatbot

  • Title: TimeSync Assistant: A Trust-Centered Scheduling Chatbot
  • Introduction: This artifact demonstrates the integration of technical skills and interpersonal values by designing a user-friendly chatbot that reflects the Eight Pillars of Trust. It showcases how academic knowledge and ethical principles translate into a meaningful, real-world AI product.
  • Description: The TimeSync Assistant is a chatbot developed to help users manage their daily schedules by accounting for meetings, travel time, breaks, and available time slots—all while converting events across time zones. This tool supports professionals in optimizing their day while promoting trust through clarity, compassion, and consistency.
  • Objective: The goal of this work was to design a functional and intelligent assistant that helps users manage time efficiently while incorporating ethical values that foster trust, collaboration, and user well-being in both academic and professional contexts.
  • Process:
    1. Team Formation – Collaborated with peers to define chatbot goals and scope.
    2. Design Planning – Mapped out use cases such as time zone conversion, travel time logic, and break scheduling.
    3. Rule Implementation – Applied logical conditions for travel and meeting overlaps based on chatbot inputs.
    4. Message Crafting – Wrote friendly, helpful prompts to support user interaction and trust-building.
    5. Demo Testing – Ran a sample schedule to validate bot behavior and ensure smooth, consistent output.
    6. Documentation – Created user documentation to explain functionality, logic, and value.
  • Tools/Technologies Used:
    • Python (logic flow and rule-based decision making)
    • Chatbot Platform (e.g., Mizou)
    • Canva (for visual presentation and UI mockups)
    • Google Docs (collaborative writing and planning)
    • Time Zone APIs (city-to-time mapping and conversion)
  • Value Proposition:
    • Unique Value: Contributed to both technical logic and ethical alignment by embedding the Eight Pillars of Trust (e.g., adding break logic for compassion, ensuring clear outputs for clarity).
    • Relevance: This project bridges technical development with human-centered design—vital for AI applications in real-world business and healthcare environments.
  • References:
    • Chatbot Demo
    • Eight Pillars of Trust Framework (Academic vs. Professional Context)
